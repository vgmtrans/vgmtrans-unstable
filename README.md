This repository hosts the latest unstable builds of **VGMTrans** for Windows, macOS, and Linux.

> [!WARNING]
> These builds are automated prereleases. They may contain regressions, incomplete features, or new bugs that are not present in the latest stable release.

You are welcome to use unstable builds to try new features before they are promoted to a stable release.

## What Is Here

- The latest unstable build is published on the releases page of this repository.
- Builds are generated automatically from the main [`vgmtrans/vgmtrans`](https://github.com/vgmtrans/vgmtrans) repository.

## Looking For Stable Releases?

Use the main VGMTrans releases page instead:

- [Stable releases](https://github.com/vgmtrans/vgmtrans/releases)

## Reporting Problems

If you hit a bug in an unstable build, please report it in the main project repository:

- [Issue tracker](https://github.com/vgmtrans/vgmtrans/issues)

When possible, include:

- the unstable build version shown on the release page
- your platform
- the file that triggered the problem
- steps to reproduce the issue
