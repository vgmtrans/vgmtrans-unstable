
# VGMTrans Preview

[![Download preview build](https://img.shields.io/badge/Download-Latest%20Preview-f97316?style=for-the-badge)](https://github.com/vgmtrans/vgmtrans-preview/releases/latest)
[![Main repository](https://img.shields.io/badge/Source%20code-vgmtrans%2Fvgmtrans-1f6feb?style=for-the-badge)](https://github.com/vgmtrans/vgmtrans)

`vgmtrans-preview` is the preview channel for **VGMTrans**. It publishes the latest automated builds for Windows, macOS, and Linux from the main [`vgmtrans/vgmtrans`](https://github.com/vgmtrans/vgmtrans) repository so you can try incoming fixes and features before the next stable release.

> [!WARNING]
> These are test builds and may be less stable than the [current](https://github.com/vgmtrans/vgmtrans/releases) VGMTrans release.
> They may include bugs, but they can also contain new features and fixes not yet available in the latest stable release.

### What This Repository Is For

- Previewing upcoming changes before they ship in a stable release
- Downloading builds without being signed in to a GitHub account
- Testing bug fixes and features and reporting feedback early

If you run into a bug in a Preview build, please open an issue in the main repository's [issue tracker](https://github.com/vgmtrans/vgmtrans/issues).
